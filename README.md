# Binary Tree Visualiser

A simple binary tree visualiser written in C.

`Under development`

# Usage
```sh
$ git clone https://github.com/anirudhbhashyam/binary-tree-visualiser
$ ./pack.sh binary_tree.c
$ dot -T<format> bst.dot > bst.<format>
```

# Dependencies
- `gcc`
- `dot (graphviz)`
